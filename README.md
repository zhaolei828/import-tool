# timu-tool
>1、题干
>
>  规则：
>
>  数字+.|． and previousElement是空行。
>
>  提取规则：
>  1、题干
>  2、题干与选项之间的内容
>  3、题干与小题之间的内容
>
>2、找答案文档
>
>	规则：必须是docx
>
>	题目和答案分开在两个word文档中，其中一个文档名称为“\*\*试题”，则这个文档为试题文档，里面都是试题；另一个文档名称为“\*\*答案”或“\*\*答案\*\*”。
>
>	所以用 “试题”去截取题目的那个文档名，然后去找对应的答案文档。
>
